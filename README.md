## Run Locally

Clone the project

```bash
  git clone https://github.com/betanyeli/simpli-note.git
```

Go to the project directory

```bash
  cd sympz-app
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```

## Running Tests

To run tests, run the following command

```bash
  yarn test
```

Or run linter

```bash
  yarn lint
```
